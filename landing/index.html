Architecture

One library.
Two worlds connected.

PhysLink is a thin but complete adapter layer. It handles all the plumbing between the physics world and the RL world so you can focus on the science.

Five primitives

Everything you need.
Nothing you don't.

🩺 Zero-friction diagnostics

One call scans your entire environment — Python, PyTorch, CUDA, VRAM, Colab session time. Returns a Go/No-Go verdict in under 15 seconds with one actionable fix per failed check. No more silent OOM at step 8 000.

physlink.doctor() # → GO ✓ | T4 · 14.8 GB · ~8h remaining
📐 Universal Space API

Define observation and action spaces in one line. Immediate type/dimension validation on creation. Each object exposes .explain() for normalization metadata.

ObservationSpace.from_proprioception(
  joints=7, include_velocity=True
)
⚡ Auto-checkpoint

Safetensors checkpoints every N steps with embedded metadata. Colab session disconnect? PhysLink resumes from the last saved checkpoint automatically. No lost GPU hours.

adapter.fit(traj, steps=10_000)
# → step_1000.safetensors saved
⚖️ Physical compliance

Register physical invariants as plain Python callables. Hard mode rejects trajectories; soft mode penalizes loss. Structured violation reports with trajectory-level diagnostics.

physlink.register_invariant(
  adapter, "energy", fn=energy_fn,
  tolerance=0.05, mode="hard"
)
🎬 Triptych visualization

Three-panel GIF — Imagination, Real, Difference — renders inline in Colab in under 10 seconds. The Time-to-Science callout computes elapsed time vs a from-scratch baseline.

adapter.visualize(trajectories)
# → triptych.gif renders inline
Quick start

Five steps.
Full pipeline.

From a blank Colab session to a validated, exported adapted model.

quickstart.py 
import physlink

# 1 — Verify environment (15 seconds)
physlink.doctor()

# 2 — Define robot spaces
obs = physlink.ObservationSpace.from_proprioception(
    joints=7,
    include_velocity=True,
)
act = physlink.ActionSpace.continuous(
    dims=7,
    bounds=(-1.0, 1.0),
)

# 3 — Adapt to DreamerV3
adapter = physlink.DreamerV3Adapter(obs, act)
adapter.fit(trajectories, steps=10_000)

# 4 — Register physical invariant & validate
physlink.register_invariant(
    adapter, "energy",
    fn=lambda traj: compute_energy(traj),
    tolerance=0.05,
)
report = adapter.compliance_report()
report.plot()   # inline Colab histogram

# 5 — Visualise & export
adapter.visualize(trajectories)   # triptych GIF
adapter.export("./run-2026/")
physlink.doctor()
